# LegallySahi
⚖️ Legally Sahi

Simple Legal Guidance for Everyone

Legally Sahi is an AI-powered legal assistant designed to help users understand legal issues in simple language. It provides structured, practical guidance for common legal problems in India.

---

🚀 Features

- 🤖 AI Legal Chatbot
  Ask legal questions and get clear, structured answers

- 📋 Structured Output
  Every response includes:
  
  - Issue
  - Explanation
  - Actionable Steps
  - Warning (if any)
  - Confidence Level

- 🧠 Smart Query Detection
  Filters non-legal queries and keeps the system focused

- ⚠️ Risk & Scam Awareness
  Highlights potential fraud or unsafe situations

- 💾 History Tracking
  Saves past queries and responses locally

---

🛠️ Tech Stack

- Python
- OpenAI API
- dotenv (for secure API key handling)

---

⚙️ Setup Instructions

1. Clone the repository

git clone https://github.com/your-username/legally-sahi.git
cd legally-sahi

2. Install dependencies

pip install -r requirements.txt

3. Create ".env" file

Create a ".env" file in the root directory and add:

OPENAI_API_KEY=your_api_key_here

4. Run the app

python app.py

---

🧪 Example Queries

Try these:

- Landlord not returning deposit
- Online scam refund issue
- Police FIR not being registered
- Fake challan received

---

🧠 How It Works

1. User enters a legal query
2. System checks if the query is legal-related
3. AI processes the request using structured prompts
4. Response is returned in a clean, readable format
